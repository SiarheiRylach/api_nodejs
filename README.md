# api_nodejs

 Simple Api with express.js</br>

***Prerequisites***:</br>
  1. Install nodejs to your OS and set all environment variables for node js.</br>
  2. Use cmd command "npm i" from root folder.</br>
  3. Run mongo server.
  
***Getting Started***</br>
    Use cmd command "node server" from root folder .</br>
